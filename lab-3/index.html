ITC 505 WEB TECHNOLOGY LAB 3 - SERVING FONTS

  • HTML Structure:
    • The document starts with the usual !DOCTYPE html declaration, followed by the opening html tag with the lang attribute set to "en".
    • Inside the head section, meta tags for character set and viewport are defined. Additionally, the title of the webpage is set to "ITC 505 WT Lab 3 - Serving Fonts Example".
  • CSS Styling:
    • The CSS block starts after the closing head tag. It first imports multiple fonts from Google Fonts using the @import rule. The fonts imported are "Nosifer", "ZCOOL KuaiLe", "Shojumaru", and "Architects Daughter".
    • CSS rules are defined to apply these fonts to specific elements:
      • The container class sets margins to create spacing around the content.
      • The heading ID styles the main heading with the "Nosifer" font, cursive style, a font size of 2.1rem, and a margin bottom of 20px.
      • The sub_heading class styles subheadings with the "Shojumaru" font, cursive style, and a font size of 1.5rem with increased line height.
      • The content class styles regular content with the "ZCOOL KuaiLe" font, cursive style, a font size of 1.0rem, and a line height of 1.6.
      • The sub-content class styles additional content with the "Architects Daughter" font and a font size of 1.0rem.
  • Content:
    • Inside the body section, there's a div element with the class "container". This is where the main content resides.
    • The content is organized into an unordered list (ul) with list items (li). Each list item contains an explanation of different aspects of the code, such as HTML structure and CSS styling. Sub-lists are used for detailed explanations under each main point.
  • Footer:
    • At the end of the body section, there's a footer element. Inside it, there's a paragraph displaying the last modified date of the document. This is achieved using JavaScript, where the script retrieves the last modified date and updates the content of the span element with the id "lastModified".
