Tic Tac Toe Game

Click "Start Game" to begin

Addendum: In the JavaScript file, I implemented a Tic-Tac-Toe game. Players take turns clicking on the cells to place their marks ('X' or 'O'). The game checks after each move if a player has won. If all cells are filled and no winner is found, the game ends in a tie. The "Start Game" button initiates the game, and the "Reset Game" button restarts it.

I encountered challenges in correctly detecting winning combinations and resetting the game state. These issues were addressed by refining the win-checking algorithm and improving the game reset functionality.
